# webT

It is a web-based linux Terminal where you can learn basic commands .
<br>

<hr>

## How to Use :computer:

1. Download or clone the repo.
2. Move to the folder.
3. Run "npm install".
4. Run "npm start".

## Technologies Used :hammer_and_wrench:

1. ReactJs.
2. JavaScript & JqueryTerminal.
